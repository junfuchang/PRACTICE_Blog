## blog-next-web

### 项目介绍

**博客前台页面展示**

使用`Nest`脚手架搭建的`React`项目：使用`Antd`组件库完成界面设计与搭建，配合`marked`、`markdown-navbar`、`highlight`等插件完成`Markdown`文章与目录的展示，代码块高亮等功能。
